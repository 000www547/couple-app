# 亲密关系日常分享微信小程序

一款专为亲密关系设计的日常分享小程序，可以记录纪念日天数、发布甜蜜时刻、管理心愿清单。

## 功能特性

- **纪念日倒计时** - 记录在一起的天数、对方生日、各种纪念日
- **甜蜜时刻** - 发布图文动态，记录甜蜜回忆
- **心愿清单** - 记录和完成共同的心愿
- **个人中心** - 编辑资料、绑定另一半

## 技术栈

- 微信小程序
- 微信云开发（云数据库 + 云函数）
- Vant Weapp UI组件库
- 主题色：粉色 #FFB6C1、蓝色 #87CEEB、黄色 #FFD700

## 快速开始

### 1. 创建云开发环境
- 打开微信开发者工具
- 点击「云开发」按钮，创建环境（环境ID记住后面需要用到）

### 2. 配置项目
- 打开 `app.js`，将 `env: 'your-env-id'` 改为你的云开发环境ID
- 打开 `project.config.json`，将 `appid` 改为你的小程序AppID

### 3. 创建数据库集合
在云开发控制台创建以下集合：
- `users` - 用户表
- `anniversaries` - 纪念日表
- `moments` - 甜蜜时刻表
- `wishes` - 心愿清单表

### 4. 部署云函数
在 `cloudfunctions` 目录上右键，选择「上传并部署：云端安装依赖」
- 需要分别部署 `login`, `moments`, `wishes` 三个云函数

### 5. 安装Vant Weapp
```
npm install @vant/weapp --save
```

### 6. 编译运行
在微信开发者工具中点击「编译」即可预览

## 使用说明

### 首次使用
1. 进入「我的」页面，点击「编辑资料」设置昵称
2. 设置「纪念日」，首页会显示恋爱天数
3. 点击「绑定亲密关系」，让对方生成邀请码后绑定

### 发布甜蜜时刻
1. 进入「甜蜜时刻」页面
2. 点击右下角「+」按钮
3. 输入文字内容，选择图片
4. 点击「发布」

### 管理心愿清单
1. 进入「心愿清单」页面
2. 点击右下角「+」添加心愿
3. 可以选择「设为共同心愿」
4. 点击复选框标记完成

## 注意事项

- 本项目使用微信云开发，需要开通云开发功能
- 图片存储在云存储中
- 请确保小程序合法合规使用
